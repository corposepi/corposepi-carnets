🔥 Configurar Firebase
Para que los carnets funcionen desde cualquier dispositivo necesitas conectar Firebase (base de datos gratuita). Solo se hace una vez.
1
Ve a console.firebase.google.com → crea un proyecto gratis → activa Realtime Database → reglas: ".read": true, ".write": true
2
En Firebase → Configuración → Tus apps → Web → copia el objeto firebaseConfig
3
Pega la URL de la base de datos abajo (termina en firebaseio.com)
4
Pega tu API Key de Firebase
5
Pega el Project ID
Firebase es gratuito hasta 1GB de datos y 10GB de transferencia al mes — más que suficiente para CORPOSEPI.
